# Universal Tool Warehouse — Glossary (One Pager)
Date: 2025-09-17

Quick-reference pulled from the Universal Tool Warehouse Master v5.0. For full context, see the master doc.

## Key Acronyms
- **DMAIC** — Define, Measure, Analyze, Improve, Control (gated path D1→C3)
- **DMA-DV** — Define, Measure, Analyze, Design, Verify (DFSS variant for new projects)
- **PRD** — Product Requirements Document (what/why/acceptance)
- **ACP-Tech** — Agentic Communication Protocol (technical milestones)
- **ACP-Biz** — Business-Process Approval & Change Protocol (commercial lifecycle)
- **OSS** — Open Source Software
- **SaaS** — Software as a Service

## Vocabulary
- **AI Agent Framework** — Multi-agent orchestration systems for autonomous workflows.
- **ACP** — Auditable Communication Protocol family used to coordinate and verify work.
- **ACP v1.0** — ICAR Audit model: Intent, Context, Action, Result.
- **ACP-Biz** — Business-Process Approval & Change Protocol — event rail for proposals, quotes, POs, SoWs, change orders, invoices, payments, refunds.
- **ACP-Tech** — Agentic Communication Protocol (Technical) — event rail for technical milestones, tests, security audits, and delivery.
- **Agent Framework** — Software systems enabling autonomous AI task execution and coordination.
- **API Framework** — Tools for building application programming interfaces efficiently.
- **Banned word** — Never use "comprehensive"; always substitute with a precise synonym.
- **CLI Tool** — Command Line Interface utilities for developer workflows.
- **Classification** — Tool categorization system (Agent Framework, ML Framework, Web Framework, etc.).
- **Cloud Platform** — Serverless and container-based deployment environments.
- **Confidence** — Certainty score; <0.85 triggers QA or re-plan.
- **Container** — Lightweight, portable development and deployment environments.
- **Data Processing** — Libraries for high-performance data manipulation and analysis.
- **Development Container** — Isolated environments for consistent development across teams.
- **Development Tool** — Utilities that enhance developer productivity and code quality.
- **DoR / DoD** — Definition of Ready / Definition of Done (phase gates).
- **Elite Tools** — 5-star rated tools representing best-in-class solutions.
- **Entry+** — Tier access level for enhanced features beyond basic tools.
- **Expert** — domain lead, signs off.
- **FDH** — Foster, Develop, Hone. Self-evolution cycle for continuous improvement.

- **Intermediate** — executes builds and deliverables.
- **Intern** — research, documentation, validation.

- **OPA/Gatekeeper** — Policy-as-code enforcement to allow or block transitions and deployments.
- **Orchestration & AI Core** — Central category for AI coordination and decision-making tools.

- **Phased Capture** — Attach self-evolution notes at the end of each DMAIC/DMADV gate.

- **Tool Shelf** — Catalog of approved tools/integrations for intelligent selection at build time.
- **Trending Tools** — Recently updated or newly popular tools gaining adoption.
- **UI Framework** — User interface libraries for building interactive applications.
- **UI/Voice & User Experience** — Tools for creating engaging user interfaces and voice interactions.
- **Universal Tool Warehouse** — Complete inventory system for AI-native development tools.

- **Vector Store** — Databases optimized for AI embeddings and semantic search.
- **Version** — Specific software version number or release identifier.
- **Web Framework** — Frameworks for building web applications and services.

## Security Tiers
- **All Tiers** — Basic development tools accessible to all users
- **Entry+** — Enhanced frameworks and APIs with basic restrictions

- **Superior** — Premium AI models and enterprise tools with usage-based billing

## Version Control Philosophy
- **Latest** — Always use most recent stable version for security and performance
- **Specific Versions** — Pin critical dependencies for reproducible builds
- **Alpha/Beta** — Early access to major framework updates (LangChain 1.0, LangGraph 1.0)
- **September 2025** — Latest research cycle with 14 major tool updates identified
