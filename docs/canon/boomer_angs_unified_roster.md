# Boomer_Angs — Unified Roster

**The House of ANG is the official home of all Boomer_Angs.** This document
is the single reconciled roster across every prior source: the vFinal
ACHEEVY Agent Logic, the Charter-Aware Refresh, the House of ANG v1.1, the
Tool Warehouse v4, session memory additions, and the Chicken Hawk fleet.

underscores. `Chicken Hawk` is two words. Any CamelCase variant
(`ResearchAng`, `CodeAng`, `RLAng`) is a bug, not canon — the Tool Warehouse
v4 used the CamelCase form and was wrong. Fix on sight.
