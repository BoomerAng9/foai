# One Direction Decision

## Decision statement

Coastal Brewing will use a single orchestration design:

**ACHEEVY routes every task. Feynman verifies evidence. NVIDIA produces low-cost model work. Chicken_Hawk checks risk. Hermes remembers. OpenClaw executes only when allowed.**

## Why this is one direction

The two prior packages are not competing architectures.

- The Feynman path answers: **Can we prove this?**
- The NVIDIA path answers: **Can we generate or classify this cheaply?**
- ACHEEVY answers: **Which path should this task take?**

## Build rule

Every task must become a task packet with:

- objective
- department owner
- route
- risk level
- evidence requirement
- approval requirement
- output destination
- receipt location

No free-floating agent activity.
