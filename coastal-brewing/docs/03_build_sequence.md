# Build Sequence

## Day 1: Foundation

- Create repo.
- Load agent prompts.
- Configure environment variables.
- Create AuditLedger database tables.
- Run smoke tests.

## Day 2: Commerce shell

- Create Shopify or storefront shell.
- Add starter coffee, tea, matcha, bundle, and subscription SKUs.
- Add draft policies.
- Keep claims conservative until supplier proof is received.

## Day 3: Supplier and claims

- Send supplier verification request.
- Create Feynman due-diligence tickets.
- Create claim receipt templates.
- Lock approved language.

## Day 4: Automation spine

- Import n8n workflows.
- Test support classification.
- Test daily digest.
- Test claim verification gate.
- Test content generation queue.

## Day 5: Launch prep

- Build first 30 days of content using NVIDIA drafts plus Chicken_Hawk review.
- Create welcome and post-purchase email flows.
- Create first wholesale prospect list through Feynman research.

## Day 6-7: Controlled launch

- Launch only approved SKUs and approved claims.
- Monitor receipts daily.
- Keep ad spend approval-gated.
