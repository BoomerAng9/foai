# Unified Architecture

```text
Owner
  -> ACHEEVY Orchestrator
    -> Task Packet Builder
      -> Route Classifier
        -> Feynman Research Cell
        -> NVIDIA Model Router
        -> Premium/Owner Lane
    -> Boomer_Ang Department
    -> Chicken_Hawk Review
    -> Hermes Receipt
    -> OpenClaw Action
```

## System roles

### ACHEEVY

Single operating director. It accepts goals, creates task packets, assigns departments, chooses routes, and checks that receipts are written.

### Feynman Research Cell

Research and verification office. It handles claims, supplier evidence, market intelligence, competitor research, and product proof.

### NVIDIA Model Router

Low-cost worker lane. It handles repeatable text generation, classification, summarization, brainstorming, clustering, and draft variants.

### Boomer_Angs

Departments:

- Boomer_Ops
- Boomer_CX
- Boomer_Marketing
- Boomer_Growth
- Boomer_Finance
- Boomer_Quality

### Chicken_Hawk

Risk checker. It reviews outputs before execution or publication.

### Hermes

Memory and receipts. It stores task packets, model-call receipts, research receipts, approval records, and action logs.

### OpenClaw

Executor. It only performs allowed actions and must check approval state before acting.
