# Storefront Path

Use Shopify first unless the owner intentionally chooses a no-Shopify build.

This folder is intentionally light because the company operating system should not be blocked by custom storefront code. The first version should validate demand using supplier fulfillment and established ecommerce infrastructure.

When ready, add either:

- Shopify theme customization files, or
- a Next.js frontend connected to Shopify Storefront API.
