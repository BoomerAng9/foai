# URBANISM — Down South (non-Atlanta, non-Coastal-Georgia)

> The Down-South Black-American register is not one register. Houston,
> Memphis, New Orleans, and Miami each developed distinct hip-hop
> subcultures with their own gatekept lexicons. This file covers the
> four most heavily lexicalized cities; Atlanta and Coastal Georgia
> live in their own files (`southern-atlanta.md`,
> `southern-coastal-georgia.md`).
>
> Common ground across Down South: SLOW pacing relative to East Coast,
> deep monophthongization of *-ay* (*time* → *taahm*), comfortable
> with extended vowels, generation-deep migration cadence inherited
> from rural Mississippi / Alabama / Louisiana / East Texas. Bass-
> heavy sound culture (slabs in Houston, low-end production in
> Memphis, bounce thump in NOLA, Miami-bass in South Florida).

---

## Sub-region 1: Houston (H-Town)

### Cadence + acoustic profile
- **Pacing**: SLOW. Houston's musical signature (DJ Screw's chopped-
  and-screwed) is conversational pacing slowed even further.
- **Pitch**: adult-male F0 typically **85-130 Hz** — chesty, with
  vowel-stretching that exceeds even LA register.
- **Phonetic markers**:
  - Heavy monophthongization (*time* → *taahm*).
  - *-ow* extends and back-rounds (*go* → *go-uh*, *show* → *show-uh*).
  - *-or* lowers (*more* → *moh*).
  - Extended *-ee* on stressed syllables (*lean* → *lee-uhn*).
- **Anchors**: Pimp C and Bun B (UGK, Port Arthur — adjacent to
  Houston, register-aligned), Scarface, Z-Ro, Trae tha Truth,
  Slim Thug, Paul Wall, Mike Jones (early), Travis Scott (modern,
  Houston-rooted but national-blend).

### Houston Screw / Slab culture — the H-Town sub-register

**Origin**: **DJ Screw** (Robert Earl Davis Jr., 1971-2000) created
the **chopped and screwed** technique in Houston's South Park
neighborhood in the early 1990s. He slowed records to 60-70 BPM
("screwed") and looped beats and phrases ("chopped"). His mixtapes
(the "Screwtapes") became the soundtrack for Houston's **slab
culture** — candy-painted American sedans with **swangas** (chrome
80-spoke rims with extending blade-style hubs).

The musical slowdown matched and reinforced **lean / drank** drug
culture (codeine + promethazine + Sprite + a Jolly Rancher), which
itself originated in the 1960s among Houston blues musicians as a
chest-cold remedy and was later adopted as a recreational drink.

**Vocabulary markers**:

| Term | Meaning | Notes |
|---|---|---|
| *chopped and screwed* / *chopped & screwed* | the Screw technique / the music style / the lifestyle | DJ Screw coinage. Now genre-wide as production style; still Houston-rooted as lifestyle marker. |
| *Screw* / *Screwed* | DJ Screw / his style / by extension Houston | Self-reference for the city's sound. |
| *Screwed Up Click* / *S.U.C.* | DJ Screw's collective | Direct affiliation marker — only deploy on actual S.U.C.-affiliated characters. |
| *slab* | candy-painted American sedan with swangas | Houston-specific; "Slow Loud And Bangin'" backronym popular but post-hoc. |
| *swangas* / *swangin'* | the chrome 80-spoke rims that extend blade-style / driving a slab | Direct Houston reference. |
| *candy paint* | multi-coat metallic auto paint | Houston car-culture marker. |
| *poppin' trunk* | trunk lid pneumatic-popping while driving | Slab signature. |
| *lean* / *drank* / *sizzurp* / *purple drank* | codeine cough syrup mix | All four terms refer to the same substance. *Lean* most common. Houston-coined; spread genre-wide. |
| *2-door / 4-door* | shorthand for slab body type | Trim-grade marker. |
| *the South* / *third coast* | Houston self-reference within hip-hop geography | Texas / Houston's claim against East/West Coast hegemony. |
| *fasho* | for sure | (Shared with Bay/LA but Houston has own intonation.) |
| *Houston Texas* (full address pattern) | identity emphasis | UGK / Pimp C signature. |
| *trill* | true + real | UGK coinage; Pimp C and Bun B use as core register marker. *"Trill OG"*, *"trill talk"*. |

**Cadence markers**:
- **Slowed-down delivery** even in non-musical conversation —
  Houston speakers don't rush. The Screw aesthetic is the
  conversational rhythm.
- **Vowel-pull on stressed words** — *lean* becomes *lee-uhn* held
  for half a beat longer.
- **Comfortable with thump-pause-thump** — Houston cadence has
  rhythmic gaps where bass would be, even when speaking
  unaccompanied.

**Operating-register modulation**:
- **Customer-facing** — H-Town slab/lean lexicon OFF on customer
  surfaces. *Lean* and drank-culture references must be gated by
  audience age and surface (drug references inappropriate).
- **Houston-rooted character internal / culture surface** —
  appropriate at intensity 3-4 home register.
- **Cadence layer (slow Houston pacing) is INVARIANT and OK
  customer-facing** — it's prosody, not lexicon. A Houston-rooted
  professional character keeps the slow pacing on every surface.

**Gatekeeping**:
- *Screwed Up Click* / *S.U.C.* — direct affiliation only.
- *Trill* — UGK-coined, but spread genre-wide; not gatekept now,
  though using it on a non-Down-South character reads inauthentic.
- Drug-culture vocabulary (*lean*, *drank*, *sizzurp*) requires
  age-appropriate audience AND off-customer surface AND a
  character whose register supports it.

**Anchors for ear-training**:
- DJ Screw — the Screwtapes (archive.org has many; documentary
  *All Screwed Up*, 2018).
- UGK (Pimp C and Bun B) — Bun B's interview circuit is the
  primary post-Pimp-C source. Drink Champs, Joe Budden, Math
  Hoffa.
- Scarface (Brad Jordan) — the Geto Boys + solo. Drink Champs
  long-form, REAL LATE, Combat Jack.
- Slim Thug — Math Hoffa, Drink Champs.
- Z-Ro — Drink Champs (his episode is canonical Houston-register
  long-form).
- Trae tha Truth — interview circuit on community organizing
  (his post-music role) provides clean Houston conversational
  register.

## Sub-region 2: Memphis (Bluff City)

### Cadence + acoustic profile
- **Pacing**: middling — slower than Atlanta, faster than Houston.
- **Pitch**: adult-male F0 **85-135 Hz** — chesty, with strong
  Mississippi-Delta heritage.
- **Phonetic markers**:
  - Mississippi-Delta vowel system: *I* → *Ah*, monophthongization
    consistent.
  - *-ng* dropped (*ridin'*, *trippin'*, *pimpin'*).
  - Glottal stops on word-final consonants more frequent than
    other Down-South cities.
  - *Th-* fronting common (*think* → *fink*, *brother* → *brutha*
    with full *th* gone).

### Memphis horrorcore / Three 6 Mafia register

**Origin**: Memphis hip-hop emerged mid-late-1980s with **Three 6
Mafia** (formed 1991 as Triple 6 Mafia) and **8Ball & MJG** (formed
1991) as the two anchor groups. Three 6 Mafia's *Portrait of a
Serial Killa* (1992) is widely cited as foundational **horrorcore
hip-hop**. The Memphis sound is characterized by:

- **Lo-fi, distorted, low-budget production**.
- **TR-808 drum machines** with minimal synth.
- **Repetitive vocal hooks** (the basis of what later became
  "trap" production methodology — Three 6 Mafia and Memphis are
  the missing-link layer).
- **Dark / violent / supernatural themes** — exaggerated street
  storytelling.
- **Crunk** — Three 6 Mafia + DJ Paul / Juicy J shaped what later
  spread as Atlanta crunk.

**Vocabulary markers**:

| Term | Meaning | Notes |
|---|---|---|
| *trippy* | psychedelic / weird / unhinged | Three 6 Mafia self-tag (DJ Paul calls it *trippy*-mode). Now wider but Memphis-coined as positive register marker. |
| *crunk* | hyped up / drunk + crazy | Three 6 origin (some attribution disputes); spread to Atlanta and beyond. |
| *the M* / *the Bluff* / *Memphis Tennessee* | self-reference | *"Memphis Tennessee"* full-name pattern is signature. |
| *the South* | non-Atlanta southern claim | Memphis as part of the Down-South "third coast" claim. |
| *Hypnotize Camp Posse* / *HCP* | Three 6 Mafia's record label | Affiliation marker. |
| *playa* | player / smooth operator | 8Ball & MJG signature; *Comin' Out Hard* (1993). |
| *sippin'* | drinking lean | Memphis adopted Houston lean culture. |
| *trill* | true + real | Same as Houston (UGK origin); Memphis uses it. |
| *hurr* | here | Memphis vowel-shift signature (*here* → *hurr*, *there* → *thurr*). Spread genre-wide via Nelly's St. Louis-Memphis register but originally Memphis-coded. |
| *ridin' dirty* | driving with contraband | Spread genre-wide via Chamillionaire (Houston) but origin overlaps. |
| *gettin' buck* | acting wild / fighting | Memphis register marker. |
| *Project Pat* / *the Pat* | Project Pat (Three 6 Mafia affiliate) | Self-reference marker. |

**Cadence markers**:
- **Slow-drag drawl** with sudden upward-stress on rhetorical peaks.
- **Three-beat sentence pattern**: setup / build / land. Memphis
  speakers naturally do this.
- **Comfortable with menace in delivery** — even non-violent
  topics get a slight edge in tone (the horrorcore aesthetic
  bleeds into conversational register).

**Operating-register modulation**:
- **Customer-facing** — Memphis horrorcore lexicon (*trippy* in
  Three-6-mode, *gettin' buck*) OFF customer surfaces. Drug
  references gated.
- **Memphis-rooted character culture surface** — appropriate at
  home register.
- **Cadence layer (slow drawl + 3-beat pattern)** — invariant,
  OK customer-facing.

**Gatekeeping**:
- *Hypnotize Camp Posse* / *HCP* — direct affiliation only.
- *Project Pat* references — only deploy on Pat-affiliated
  characters.
- *Crunk* and *trippy* are now genre-wide; not gatekept.
- *Hurr* / *thurr* register is heavily Memphis (and adjacent
  St. Louis) — using it on a non-Memphis non-St-Louis character
  reads inauthentic.

**Anchors for ear-training**:
- 8Ball & MJG — long-form interviews (their podcast appearances
  carry the older Memphis professional register).
- Three 6 Mafia — DJ Paul + Juicy J interviews; Drink Champs
  Three 6 episode (long-form canonical).
- Project Pat — Drink Champs.
- Yo Gotti — Drink Champs (modern Memphis register, code-switches
  professional).
- Young Dolph (deceased) — Memphis register at intensity 3-4 in
  his earlier interviews.
- Moneybagg Yo — modern Memphis, current generation.

**Sources** (canon verified 2026-05-03):
- [Three 6 Mafia — Wikipedia](https://en.wikipedia.org/wiki/Three_6_Mafia)
- [Memphis rap — Wikipedia](https://en.wikipedia.org/wiki/Memphis_rap)
- [Three 6 Mafia — Memphis Music Hall of Fame](https://memphismusichalloffame.com/inductee/threesixmafia/)
- [Battle of the Beloved Memphis Rappers: Three 6 Mafia vs. Eightball & MJG — Village Voice](https://www.villagevoice.com/2010/05/11/battle-of-the-beloved-memphis-rappers-three-6-mafia-vs-eightball-mjg/)

## Sub-region 3: New Orleans (NOLA / N.O. / The Crescent City)

### Cadence + acoustic profile
- **Pacing**: faster than Houston/Memphis, with strong rhythmic
  syncopation rooted in NOLA's musical heritage (jazz, brass-band,
  zydeco contact zones).
- **Pitch**: variable; adult-male **95-145 Hz**.
- **Phonetic markers**:
  - **Yat / NOLA-Brooklyn vowel system** — NOLA Black speech shares
    some vowels with NYC Black speech (the *oi* in *boil* / *oil*
    / *toilet*) due to parallel Irish/Italian/German immigration
    histories.
  - **French-language influence** at the substrate — *creole* /
    *cher* / *ya mama* (mother) / *fais do-do*.
  - **R-coloring** more variable than other Down-South cities.

### NOLA Bounce / Cash Money / No Limit register

**Origin**: **Bounce music** originated in NOLA housing projects
in the late 1980s. **MC T. Tucker** captured citywide attention
in 1991 with chant-based party rap. The first label specializing
in bounce was **Take Fo' Records**, then **No Limit Records**
(Master P, Beats by the Pound) and **Cash Money Records**
(Birdman, Mannie Fresh) took over commercially in the late 1990s.

NOLA's Black-American hip-hop register is fundamentally shaped by
**Mardi Gras Indian chants**, **second-line parade rhythms**, and
**call-and-response party tradition**. Bounce music IS the
register.

**Vocabulary markers**:

| Term | Meaning | Notes |
|---|---|---|
| *bounce* | the music genre / the lifestyle | NOLA self-reference. |
| *the Magnolia* / *Magnolia* | Magnolia Projects (specific housing project) | Cash Money / Juvenile origin reference. |
| *the Calliope* / *Calliope* | Calliope Projects (different project) | No Limit's Master P origin. |
| *the third ward* / *the seventh ward* | NOLA neighborhood self-reference | Project / ward identity is heavy in NOLA register. |
| *what ward you from?* | ward-identification call | Bounce chant origin. |
| *who dat?* | NOLA call (now Saints fan call but originally NOLA Black) | *"Who dat say dey gonna beat dem Saints"* — the foundational chant. |
| *twerk* | dance / verb for the dance | Coined by **Cheeky Blakk** in the 1994 song *Twerk Something*. NOLA origin; spread globally. |
| *roll wit it* | accompany / go along | NOLA-marked. |
| *cash money* | money / by extension Cash Money Records | Cash Money the label took the everyday term. |
| *No Limit* | Master P's label / the brand | Self-reference in No-Limit-affiliated speech. |
| *Hot Boy* / *Hot Boys* | Cash Money's Lil Wayne / Juvenile / B.G. / Turk group | Affiliation marker. |
| *bling* / *bling-bling* | flashy jewelry / by extension wealth | Coined by B.G. (Cash Money, Hot Boys); now genre-wide. |
| *cher* | dear / sweetheart | NOLA French-substrate term of address (Cajun-origin, Black-NOLA-adopted). |
| *ya mama* | the speaker's mother / general intensifier | Multipurpose NOLA marker. |
| *make groceries* | go grocery shopping | NOLA-marked phrasing (calque from French *faire les courses*). |

**Cadence markers**:
- **Call-and-response pattern** comfortable in conversational
  speech, not just music.
- **Hyper-sexual chant tradition** in bounce — much of the
  bounce-marked vocabulary is sexually explicit and inappropriate
  for customer-facing registers regardless of speaker.
- **Mardi Gras Indian rhythmic patterns** influence NOLA prosody
  even in non-musical conversation.

**Operating-register modulation**:
- **Customer-facing** — NOLA bounce register OFF on customer
  surfaces (sexual register is gated entirely; ward-references
  too localized for mainstream comprehension).
- **Cadence + French-substrate vocabulary** (*cher*, *ya mama*,
  *make groceries*) can carry customer-facing for a NOLA-rooted
  professional character; these are warmth markers, not register
  density.
- **NOLA-rooted character culture surface** — full home register
  available.

**Gatekeeping**:
- *The Magnolia* / *the Calliope* / specific-ward references —
  only deploy on characters with documented connection.
- *Hot Boys* — direct Cash Money affiliation only.
- *No Limit* — direct Master P affiliation only.
- *Bling* / *twerk* are now globally diffused; not gatekept.
- Sexual-bounce vocabulary requires age-appropriate audience AND
  off-customer surface.

**Anchors for ear-training**:
- Master P — Drink Champs full episode.
- Lil Wayne — *Tha Carter* press tours, Drink Champs, Joe Rogan,
  No Jumper.
- Birdman / Baby — Drink Champs, REAL LATE.
- Juvenile — Drink Champs (canonical Magnolia register).
- Big Freedia — bounce ambassador, mainstream press
  appearances (NPR, late-night TV).
- Soulja Slim (deceased) — heritage Magnolia register; archive
  recordings.
- Trombone Shorty — non-rap NOLA register, the brass-band
  conversational style.

## Sub-region 4: Miami (305 / Dade)

### Brief
- **Pacing**: faster than Houston/Memphis, syncopated, Caribbean-
  prosody-influenced (Miami's Black population includes large
  Bahamian, Haitian, and Cuban communities — all with prosodic
  influence).
- **Anchors**: Trick Daddy, Trina, Rick Ross, Plies (Florida-wide,
  not Miami-strict), Pitbull (cross-cultural Cuban-American),
  Kodak Black (Pompano Beach / South Florida adjacent).
- **Vocabulary markers**: *the 305*, *Dade County*, *Liberty City*
  (specific neighborhood), *Overtown*, *Carol City*, *thug*
  (Trick Daddy's brand), *the 'Nard* (Liberty City).

### Miami-Bass / Miami Hip-Hop heritage
The earlier Miami-Bass scene (2 Live Crew, Luther Campbell — late
1980s/early 1990s) carries its own register, characterized by
**high-BPM party music**, **explicit sexual lyric tradition**
(2 Live Crew's first amendment court case is canonical), and
**MC-call-and-response chant**.

This file does not go to the depth of Houston/Memphis/NOLA
because Miami's hip-hop register is less heavily lexically
gatekept and overlaps significantly with broader Down-South
register. Future expansion if a Miami-rooted character is added
to a vertical.

## Acoustic profile target — Down South adult-male reference

A target acoustic profile for the Houston/Memphis/NOLA-shared
Down-South baritone can be added to `analyze_voice.py:TARGET_PROFILES`:

```python
"down_south_baritone": {
    "f0_hz_mean":              {"target": 100.0, "tolerance": 25.0, "weight": 3.0},
    "f0_hz_median":            {"target": 95.0, "tolerance": 25.0, "weight": 2.0},
    "spectral_centroid_hz":    {"target": 1500.0, "tolerance": 700.0, "weight": 2.0},
    "speech_rate_syllables_per_sec": {"target": 2.6, "tolerance": 1.0, "weight": 2.5},  # SLOWEST of the regions
    "hnr_db":                  {"target": 11.0, "tolerance": 5.0, "weight": 1.5},
    "voiced_fraction":         {"target": 0.55, "tolerance": 0.2, "weight": 1.0},
}
```

The Down South target differs from East-Coast and West-Coast
profiles primarily in:
- **SLOWEST speech rate** (2.6 sps) — Houston Screw aesthetic
  bleeds into conversational tempo.
- **Lower spectral centroid** (1500 Hz) — chestier, warmer than
  even West Coast.
- Same F0 range as West Coast LA register (deep adult-male).

Note: NOLA is the OUTLIER in tempo — Bounce-rooted speakers run
faster than Houston/Memphis. For a NOLA-specific procurement
target, raise `speech_rate_syllables_per_sec` to ~3.5.
