# Caribbean — Group root

> Anglophone Caribbean register: Jamaican, Trinidadian, Bajan,
> Bahamian, Vincentian, Belizean, Guyanese (Caribbean-adjacent),
> plus Haitian-American (Kreyòl-substrate English). NOT a single
> register — each island/national-origin has distinct phonology,
> vocabulary, and prosody. Different from Caribbean-influenced
> Black-American register tracked under URBANISM (Bronx / Brooklyn
> heritage layer).

## Scope

In scope:
- Jamaican Patois / Jamaican English continuum.
- Trinidadian English / Trini Creole.
- Barbadian (Bajan) English.
- Bahamian English.
- Vincentian (St. Vincent) English.
- Belizean Kriol / English.
- Guyanese English (geographically South American but culturally
  Anglophone Caribbean).
- Haitian-American English (Kreyòl-substrate).
- US diaspora register: Brooklyn West Indian, Miami Caribbean,
  Boston Haitian, NYC Trinidadian.

Out of scope:
- French Caribbean (Martinique / Guadeloupe pure French) — separate
  internationalization track.
- Spanish Caribbean (Cuba, PR, DR) — lives under `latin-american/`.
- Black-American Caribbean-influenced register (Bronx Black / Brooklyn
  Black with West Indian heritage) — overlap, but tracked under
  URBANISM as the heritage layer of those neighborhoods.

## Register / intensity scale

Caribbean register sits on a **continuum scale 0-4** between Standard
English and full creole/patois:

| Level | Name | Description |
|---|---|---|
| **0** | Standard English | Caribbean speaker in formal / professional / non-Caribbean audience setting. No detectable patois. |
| **1** | Light heritage | Standard grammar with detectable rhythm + occasional vocabulary (*irie*, *boss*, *yardie*). Caribbean professional in mainstream context. |
| **2** | Conversational mesolect | Mid-continuum register: some patois grammar (e.g. zero-copula *she nice*), more vocabulary, prosody clearer. Casual conversation among Caribbean speakers. |
| **3** | Strong patois | Heavy creole grammar + vocabulary. *Mi* for *I*, *unu* for *you-plural*, *deh* (be), *fi* (for/to). Native-context register. |
| **4** | Deep creole | Full Patois / Kriol / Kreyòl. May not be intelligible to non-creole speakers. Heritage register. |

Note: this is a **continuum**, not a fixed register — speakers slide
along it within a single conversation, governed by audience and topic.

## Phonetic / cadence signatures (group-level)

(To be expanded per regional sub-file. Major patterns:)

- **Syllable-timed pacing** — like Belter Creole and like
  URBANISM Coastal-Georgia register, every syllable gets close-to-
  equal weight. Distinguishes Caribbean from stress-timed General
  American.
- **Rising-falling sentence-end intonation** — distinctive
  question/statement-end rise (often called "sing-song").
- **/th/ → /t/ or /d/** — *think* → *tink*, *that* → *dat*. Variable
  by register level.
- **/r/-elision** word-final and pre-consonantal — *here* → *hia*,
  *for* → *fa*.
- **Vowel system** — Caribbean English has different vowel positions
  than General American (*man* fronter, *not* more open).

## Gatekeeping notes

- **Patois deploy** — full Patois on a customer surface where the
  audience doesn't speak it = comprehension wall. Modulate per
  `REGISTER-MODULATION.md`.
- **Rastafari language** (Iyaric — *I-and-I*, *livity*, *Babylon*,
  *overstand*) is a religious / cultural layer; deploying it on
  non-Rasta characters reads as cosplay.
- **Dancehall lexicon** has gang-affiliation overlap (don / shotta /
  badman) — register-authenticity gates apply same as URBANISM
  drill.
- **Cross-island differences** matter — using Trini *liming* on a
  Jamaican character is wrong. Using *yardie* on a Trini character
  is wrong.

## Anti-patterns

- Don't write all Caribbean characters in Jamaican Patois. Each
  island is distinct.
- Don't render Caribbean characters as comedic "yah mon" parody.
  The actual register is far more sophisticated.
- Don't conflate Haitian (Kreyòl-substrate) with Anglophone
  Caribbean — French-substrate is fundamentally different prosody.
- Don't write Caribbean speakers at level 4 (deep creole) on
  mainstream surfaces — comprehension fails.
- Don't deploy Rastafari language as a "spiritual feel" flavor.
  It's a religious register with theological weight.
