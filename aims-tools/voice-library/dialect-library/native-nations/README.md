# Native Nations — Group root

> Indigenous American (First Nations / Native American / Alaska
> Native / Native Hawaiian / Indigenous Mexican-American) Anglophone
> register. **Critical naming distinction**: "Indigenous" in this
> library refers to URBANISM (Black Americans, per owner directive).
> THIS group is named **Native Nations** to avoid the collision —
> Native American / First Nations / tribal nations / Alaska Native
> peoples are tracked here.

## Scope

In scope:
- **Federally recognized tribes in the US** — 574+ tribal nations,
  with major language families: Navajo (Diné Bizaad), Lakota /
  Dakota / Nakota (Sioux), Cherokee, Choctaw, Chickasaw, Creek
  (Muscogee), Seminole, Hopi, Zuni, Pueblo nations, Apache, Iroquois
  (Haudenosaunee — Mohawk, Oneida, Onondaga, Cayuga, Seneca,
  Tuscarora), Ojibwe / Anishinaabe, Lummi, Salish, Tlingit, Haida,
  Inupiat, Yupik, etc.
- **Canadian First Nations** — cross-border tribal continuity
  (Ojibwe, Mohawk, Cree, Inuit, etc.).
- **Native Alaskan** — Inupiat, Yupik, Tlingit, Haida, Aleut.
- **Native Hawaiian** — distinct from Pacific Islander (tracked under
  `asian-american/regional/pacific-islander.md` for now; may move
  here per cultural sovereignty argument).
- **Indigenous Mexican-American** — Zapotec, Mixtec, Mayan, Nahua
  diaspora in US (Oaxacan / Yucatecan communities in LA, North
  Carolina, NY). Cross-cuts `latin-american/`.
- **Métis** (Canadian mixed Indigenous-European) — distinct identity.
- **Pan-Indian English** — pan-tribal register that emerged from
  inter-tribal urban communities (NYC, LA, Phoenix, Denver,
  Minneapolis, Albuquerque powwow circuit register).

Out of scope:
- Indigenous languages spoken purely without English — separate
  internationalization track.
- Latin American Indigenous (Mayan, Quechua, etc.) outside US/Canada
  diaspora — separate track.

## Register / intensity scale

Native Nations register sits on a **substrate-influence scale 0-3**
plus a **tribal-specificity dimension**:

| Level | Name | Description |
|---|---|---|
| **0** | Standard professional | Native speaker, no detectable heritage substrate. Common in 2nd-gen-urban, college-educated contexts. |
| **1** | Light heritage | Detectable accent + occasional vocabulary loans (*chee* / *aoo* in Navajo, *waste* / *hau* in Lakota). Casual professional. |
| **2** | Conversational heritage | Substrate prosody clearly audible, frequent code-switch with heritage language, sentence patterns from heritage audible. |
| **3** | Heavy heritage | Strong heritage prosody dominant; reservation-resident register; first-language heritage speakers. |

Plus a **traditional vs urban dimension**:
- **Traditional / reservation-resident**: more substrate density,
  more cultural-vocabulary usage, slower pacing.
- **Urban Indian**: pan-tribal blend, more General American
  substrate, BUT carries cultural-language layer (sovereignty,
  treaty rights, AIM activism, MMIW, Land Back movement).

## Phonetic / cadence signatures (group-level)

(To be expanded per regional sub-file. Major patterns:)

- **Tone / pitch substrates** — Navajo and several Athabaskan
  languages have tone systems; tone-substrate audible in English.
- **Glottalized consonants** — many Native languages have ejective
  / glottalized consonants that surface in heritage-substrate
  English.
- **Slow-deliberate cadence** — many traditional Native registers
  have slower-than-General-American pacing, with comfortable silence.
  This is **culturally meaningful** — pause is conversational space,
  not awkward.
- **Discourse markers** — *ahéhee'* (Navajo thank you), *hau* /
  *waste* (Lakota greeting / good), *miigwech* (Anishinaabe thank
  you), *aho* (pan-tribal closing).
- **Storytelling cadence** — many Native registers preserve
  oral-tradition storytelling cadence even in casual conversation.

## Gatekeeping notes

- **Tribal sovereignty** — every tribe is a sovereign nation. Naming,
  identity, and cultural-vocabulary deploy must respect this.
- **574+ federally recognized tribes** — no single Native register.
  Don't deploy "Native American" as if it's homogeneous.
- **Tribal enrollment vs descent** — politically loaded; tracked
  by tribes, not by external groups.
- **Ceremonial language** — many tribes have ceremonial vocabulary
  that is sacred and **not for casual / fictional / commercial
  use**. Sweat lodge language, Sun Dance language, ceremonial
  songs are off-limits for character voicing.
- **Pretendian** problem — non-Native characters claiming Native
  identity is a serious cultural-appropriation issue. Render
  Native characters with documented tribal-identity; do not
  generate characters who casually claim Native ancestry.
- **MMIW (Missing and Murdered Indigenous Women)** is a real
  ongoing crisis; cultural-vocabulary references to it are
  somber, never casual.
- **Land Back** + **#NoDAPL** + **Sovereignty** are political
  positions; render character voices on register signals,
  not political signaling unless explicitly authorized.

## Anti-patterns

- Don't render Native characters with "noble savage" / "wise
  shaman" / "stoic warrior" tropes. Brand-damaging and
  stereotype-reinforcing.
- Don't deploy ceremonial language as flavor.
- Don't write "Indian English" as broken English. Heritage-substrate
  Native speech is its own legitimate register.
- Don't conflate tribes — Cherokee is not Navajo is not Lakota
  is not Cree.
- Don't write urban-Indian register as "lost their culture."
  Pan-tribal urban-Indian community has its own rich cultural-
  language tradition.
- Don't use *spirit animal* / *tribe* (as in friend group) /
  *powwow* (as in casual gathering) / *low man on the totem pole*
  / *Indian giver* / *off the reservation* — all are appropriated
  / pejorative usages.
