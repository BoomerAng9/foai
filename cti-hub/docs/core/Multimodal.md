# Multimodal

The GRAMMAR interaction plane leverages multiple senses (text, visual, voice).

1. Voice Input (STT) via Groq Whisper adapter logic.
2. Voice Output (TTS) via ElevenLabs synthesized returns.
3. Visual Comprehension via OpenAI models tagging and parsing UI schematics.
4. Extensible Response Composer to aggregate array returns to unified client interfaces (Voice Audio blob + Transcription String).
