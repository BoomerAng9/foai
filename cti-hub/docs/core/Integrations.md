# Integrations

External data domains bridged to the Orchestrator via OpenClaw endpoints.

## Provider Examples
- Stripe (Escrow, capturing, refunds)
- OpenAI (Vision mapping, text mapping)
- OpenRouter (Inference abstraction)
- MinIO (Blob Object Storage, CDN replacements)
- Redis (Session Pub/Sub state sync)
- Postgres (Prisma persistence tables)
- NextAuth (JWT access controls)
