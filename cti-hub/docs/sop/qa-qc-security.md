# SOP: QA/QC and Security Hardening

## QA/QC Checklist
- Contracts validate.
- Unit/integration tests pass.
- ASCII artifact linked for UI-facing work.
- Design matches existing theme direction.
- Required controls behave correctly.
- Docs updated.

## Security Checklist
- Per-agent tool access enforced.
- Unsafe tool paths denied.
- No hardcoded secrets.
- Sandbox intent/profile tagged.
- Auth assumptions documented.
