## Subscription Tier Updates

### YOLO Mode (Highest Tier Only)
- Activates /yolo (Claude-style autonomous mode)
- Runs until done — no stopping for approval
- User incurs full cost
- LUC shows: estimated time, estimated cost, model options
- User understands they're paying for full autonomous execution

### Lower Tiers
- Non-autonomous: stops for approval at checkpoints
- Notifications when things need to restart
- LUC provides estimates at each gate

### LUC "Adjust" Feature
- When LUC shows cost estimate, "Adjust" offers alternative models
- Models that fit the quality/capability need but cost less
- No free models at highest tier — still paid but cheaper than Opus 4.6
- Pay-per-use: open — any model including free

### Commitment Windows (Pricing)
- 3 months: pay 3 months upfront → renew or cancel
- 6 months: pay 6 months upfront → renew or cancel
- 9 months: pay 9 months → GET 12 MONTHS
  - Each tier has monthly token allotment
  - Hit cap → pay-per-use at NEAR COST (few % markup only)
  - 9-month users get the best overage rate

### Transparency Toggle
- Always-on visibility into what models are used and costs
- At each stage of a plug: show what's being consumed
- User knows what they're buying at all times

## Add-on Features (per plug)

### Continuous Improvement
- Auto-research at every stage of every plug
- Self-evolution: finds gaps, suggests improvements, maintains
- User elects this as an add-on
- Affects token usage — user sees the impact in LUC

### Code Validation (Code_Ang + LeanStral)
- Mistral's LeanStral — open source Lean4 code validation
- Formal verification for code quality
- Code_Ang runs this on every code deliverable
- Add-on: user sees it as a security/quality package
