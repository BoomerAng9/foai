# Plug Bin Hero Tiles — Iller_Ang / Recraft V4 Generation Prompts

Generated via Open Mind creation harness applied to the `/plug` tile
problem. Defaults rejected: raw Unicode emojis as icons, generic stock-
photo compositions, AI-garbled text in frame, any generic "AI art" look.

**Model:** Recraft V4 (`recraftv3`) via Recraft's OpenAI-compatible API
**Size:** 1024x1024 square (tile-native aspect)
**Style:** `realistic_image` / `digital_illustration` cinematic concept art
**Output format:** WebP, ≤200KB target per tile after encoding

Each tile is regenerated by re-running `scripts/iller-gen-plugs.py` on
a machine with `RECRAFT_API_KEY` available in env. The canonical key
lives in the `openclaw-sop5-openclaw-1` container env on myclaw-vps.

## Composition rules (apply to all 5)

- **Square 1024x1024.** Card tiles are square; never generate 16:9.
- **Character centered or slight left-of-center.** Not hard-edged at frame.
- **Atmospheric depth.** Always establish a background context beyond
  the character — a port, a stadium, a boardroom, a war room, a
  classroom. Characters in a void are rejected.
- **Warm key + cool fill.** Each composition balances a warm key light
  (gold/amber/orange) with a cooler fill (cyan/navy/teal). Single-
  temperature lighting is rejected.
- **Full visor rule (CRITICAL).** Any Boomer_Ang character must have
  the helmet visor covering the entire face AND neck. Zero skin visible
  above the chest plate. This is Rish's explicit correction on
  Ops_Ang and CFO_Ang; it applies to EVERY Boomer_Ang going forward.
- **No AI-garbled text.** If any text appears in the frame, it must be
  ABSTRACT shapes that read as writing without being legible.
- **Iller_Ang design tokens.** bg_primary #0A0A0F, accent_gold #D4A853,
  accent_cyan #00E5CC, accent_orange #FF6B00 should appear in the
  palette hierarchy but do not override each plug's brand color.
